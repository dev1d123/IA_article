En primer lugar, se utilizará el conjunto de datos \textit{Diabetes Health Indicators}, conformado por 433\,323 registros y 20 variables relacionadas con antecedentes médicos, factores demográficos y hábitos de vida asociados con la diabetes. La selección de este conjunto de datos responde a que los estudios revisados utilizan principalmente bases de datos tabulares con indicadores clínicos y de salud para entrenar modelos de clasificación supervisada \cite{Khokhar2024,ElSofany2024}.

Posteriormente se realizará un Análisis Exploratorio de Datos (EDA), cuyo objetivo será comprender la estructura del conjunto de datos antes del entrenamiento de los modelos. Durante esta etapa se analizará la distribución de las variables, la frecuencia de cada clase, las estadísticas descriptivas, la correlación entre atributos y la presencia de posibles valores atípicos o inconsistencias. Este procedimiento permitirá determinar las necesidades reales de preprocesamiento y seleccionar las técnicas más adecuadas para el conjunto de datos \cite{Khokhar2024,Olisah2022}.

La etapa de preprocesamiento comprenderá la revisión de registros inconsistentes, la identificación de valores faltantes, la eliminación de registros duplicados en caso de existir y la normalización de las variables numéricas cuando el algoritmo empleado sea sensible a la escala de los datos. Asimismo, se evaluará el grado de desbalance entre las clases de la variable objetivo; únicamente si dicho desbalance resulta significativo se considerará la aplicación de técnicas de sobremuestreo como SMOTE, siguiendo las recomendaciones observadas en diversos trabajos recientes \cite{Abnoosian2023,Khokhar2024}.

Una vez preparados los datos, se realizará un análisis de la relevancia de las variables mediante medidas de correlación y métodos basados en la importancia de características proporcionada por modelos de árboles de decisión, como Random Forest y XGBoost. Debido a que el conjunto de datos contiene únicamente veinte variables, no se contempla inicialmente la aplicación de técnicas de reducción de dimensionalidad como PCA o ICA, aunque estas podrán evaluarse si los experimentos evidencian beneficios en el desempeño predictivo \cite{Olisah2022,Abnoosian2023}.

El conjunto de datos será dividido en subconjuntos de entrenamiento y prueba. Sobre el conjunto de entrenamiento se aplicará validación cruzada estratificada (\textit{Stratified k-Fold Cross Validation}), técnica ampliamente utilizada en la literatura debido a que proporciona una estimación más robusta del rendimiento del modelo y mantiene la proporción de las clases durante el proceso de entrenamiento \cite{Olisah2022,Abnoosian2023}.
 
La fase de entrenamiento consistirá en la comparación de cuatro algoritmos de clasificación: Regresión Logística (Logistic Regression), Random Forest, Extreme Gradient Boosting (XGBoost) y Redes Neuronales Multicapa (Multi-Layer Perceptron, MLP). Estos modelos fueron seleccionados porque representan diferentes enfoques de aprendizaje supervisado y han mostrado resultados competitivos en problemas de predicción de diabetes utilizando datos tabulares \cite{ElSofany2024,Khokhar2024}.

Con el propósito de maximizar el desempeño predictivo de cada clasificador, se realizará un proceso de optimización de hiperparámetros mediante Grid Search combinado con validación cruzada. Para Regresión Logística se evaluarán parámetros relacionados con la regularización; para Random Forest se optimizarán el número de árboles y la profundidad máxima; para XGBoost se ajustarán parámetros como la tasa de aprendizaje, el número de árboles y la profundidad máxima; mientras que para MLP se explorarán diferentes configuraciones de arquitectura, función de activación y tasa de aprendizaje \cite{Olisah2022,Abnoosian2023,ElSofany2024}.
 
Finalmente, el rendimiento de cada modelo será evaluado mediante las métricas Accuracy, Precision, Recall, F1-score y Área Bajo la Curva ROC (ROC-AUC), las cuales constituyen las métricas de evaluación más utilizadas en la literatura sobre predicción de diabetes. A partir de estos resultados se comparará el desempeño de los clasificadores y se seleccionará el modelo que presente el mejor equilibrio entre capacidad predictiva, estabilidad e interpretabilidad. En caso de que el mejor modelo corresponda a Random Forest o XGBoost, se realizará además un análisis de importancia de variables para identificar los factores con mayor influencia en la predicción del riesgo de diabetes \cite{ElSofany2024,Khokhar2024,Olisah2022}.
